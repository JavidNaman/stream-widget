# ویجت یادمان جاویدنامان ۱۴۰۴

سلام. این پروژه رو ساختیم تا استریمرها بتونن خیلی راحت و با یه ویجت شیشه‌ای و مینیمال، اسم و عکس جاویدنامان سال ۱۴۰۴ رو گوشه استریمشون نشون بدن و یادشون رو زنده نگه دارن.

## آموزش استفاده تو OBS
اضافه کردن این ویجت به استریمتون خیلی ساده‌ست:

۱. تو نرم‌افزار OBS (یا Streamlabs)، قسمت **Sources** روی آیکون **+** کلیک کنید.
۲. گزینه **Browser** رو انتخاب کنید و یه اسم دلخواه (مثلاً Yadman) براش بذارید.
۳. تو پنجره‌ای که باز میشه، تو قسمت **URL** این آدرس رو کپی کنید:
`https://JavidNaman.github.io/stream-widget/`
۴. برای اینکه سایز و ظاهر ویجت به هم نریزه، ابعاد رو دقیقاً این‌طوری تنظیم کنید:
- **Width (عرض):** `400`
- **Height (ارتفاع):** `150`

۵. دکمه OK رو بزنید. حالا ویجت رو بکشید و هر جای تصویر (پیشنهاد ما سمت چپ صفحه است) که دوست دارید قرار بدید.

## چطوری لیست رو کامل‌تر کنیم؟ (مشارکت)
شما هم می‌تونید کمک کنید تا اسم بقیه جاویدنامان رو به این ویجت اضافه کنیم. اصلاً کار پیچیده‌ای نیست:

۱. بالای همین صفحه، روی دکمه **Fork** کلیک کنید تا پروژه تو اکانت خودتون کپی بشه.
۲. تو صفحه‌ای که براتون باز میشه، فایل `data.json` رو باز کنید و روی آیکون مداد ✏️ کلیک کنید تا بتونید ویرایشش کنید.
۳. اطلاعات شخص جدید رو دقیقاً مثل الگوی زیر به آخر لیست اضافه کنید:

```json
{
    "first_name": "Name",
    "last_name": "Lastname",
    "age": "Age",
    "location": "City - Date",
    "image_url": "Image URL"
}

```

۴. **برای لینک عکس چیکار کنیم؟** برید تو تب **Issues** گیت‌هاب، عکس اون شخص رو با موس بکشید و تو کادر متن رها کنید (Drag & Drop). چند ثانیه صبر کنید تا گیت‌هاب یه لینک بهتون بده. همون لینک رو کپی کنید و بذارید جلوی `image_url`. (اون صفحه Issue رو هم بدون اینکه ذخیره کنید ببندید بره).
۵. بیاید پایین صفحه و روی دکمه سبز رنگ **Commit changes** کلیک کنید تا اطلاعات سیو بشه.
۶. برگردید به همین صفحه اصلی پروژه، برید تو تب **Pull requests** و روی **New pull request** کلیک کنید تا تغییراتی که دادید برای ما ارسال بشه و بیاد روی ویجت اصلی.

## درباره ساخت پروژه

تمام این پروژه با هوش مصنوعی گوگل راه‌اندازی شده (حتی همین متنی که الان خوندید). به همین خاطر ممکنه یه جاهایی باگ یا مشکلی وجود داشته باشه. اگه ایرادی دیدید، خیلی راحت می‌تونید خودتون اصلاحش کنید و درخواست بدید.
